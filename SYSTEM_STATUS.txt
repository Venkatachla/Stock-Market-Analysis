9. TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

  ❌ Backend Port Already in Use:
     Solution: Get-Process python | Stop-Process -Force
  
  ❌ Frontend Not Starting:
     Solution: cd frontend && npm install && npm run dev
  
  ❌ Database Issues:
     Solution: python db_viewer.py # Check
               python add_demo_users.py # Reinit
  
  ❌ Login Doesn't Work:
     Solution: Check backend logs
              Clear browser cache (Ctrl+Shift+Delete)
              Verify database: python db_viewer.py
