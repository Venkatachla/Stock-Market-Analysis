## 🚀 HOW TO START

### Quickest Way (2 minutes)
```batch
REM Windows
START.bat

REM Linux/Mac
bash START.sh
```

### Manual Way (5-7 minutes)
```bash
# Backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python -m uvicorn api.app:app --reload

# Frontend (new terminal)
cd frontend
npm install
npm run dev
```
